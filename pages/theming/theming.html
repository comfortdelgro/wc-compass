You want it to follow your style or your color scheme? It's easy!

By default, we display the layout with light theme. To display your content as dark theme, just simply add theme-dark class name to your body. To let the layout follows OS theme automatically, just add theme-os into your body.

To customize with your own color plate, you just need to create an other css file and import it under our import. With your customize css. To refer our css color variables, please refer: _style.scss
Remember to give your css higher priority by adding it after our css or giving it complex selector.

For example, if you want to override our color scheme. First let's go and create a new css file and override our css variables by these: 

    
:root {
    --background: #ffffff;
    --revese-color: #ffffff;
    --accent: #0142af;
    ...and any other variables
}

For dark theme

    
/** Follow OS Configuration **/
@media (prefers-color-scheme: dark) {
    body.theme-os {
        --background: #ffffff;
        --revese-color: #ffffff;
        --accent: #0142af;
        ...and any other variables
    }
}

/** By adding any custom class name **/
body.theme-dark {
    :root {
        --background: #ffffff;
        --revese-color: #ffffff;
        --accent: #0142af;
        ...and any other variables
    }
}
