This gutter control will align with parent element, it will update directly to parent element size.
To implement it, just place it into the element that you want to resize.
To let the other element fill the rest space of their parent please use spearate css or refer to cdg-layouts

Some content belong to left column
Here is the main content
    <div class="cdg-layout sample-gutter-content">
    <div class="cdg-group-left">
        <div class="left-col-content">Some content belong to left column</div>
        <cdg-gutter></cdg-gutter>
    </div>
    <div class="cdg-group-center cdg-flex">Here is the main content</div>
</div>

Some content belong to right column. This column has 50% maximum size, so that you can not resize it bigger than 50% of it's parent. 
    <div class="cdg-layout sample-gutter-content">
    <div class="cdg-group-center cdg-flex">Here is the main content</div>
    <div class="cdg-group-right">
        <div class="right-col-content">Some content belong to right column</div>
        <cdg-gutter place="left"></cdg-gutter>
    </div>
</div>
