Our icons are using svg files for better quality and easy to override the color.
If we build the whole images into the bundle file, our resource will be extremely large and the page load speed will be slow down. So that we would like to seprate them into a folder.
Once you've declare an icon with a name, icon component will call the api to fetch the svg file and attach them into the DOM. 

    <cdg-icon name="user" size="48"></cdg-icon>
<cdg-icon name="user" size="40"></cdg-icon>
<cdg-icon name="user" size="32"></cdg-icon>
<cdg-icon name="user" size="24"></cdg-icon>
<cdg-icon name="user" size="16"></cdg-icon>

Based on current text color, just give the paragraph a color, the icon will follow that color

This pharagraph have danger color 

    <cdg-icon name="user" style="color: var(--accent)"></cdg-icon>

<cdg-icon name="user" style="color: var(--success)"></cdg-icon>

<cdg-icon name="user" style="color: var(--warning)"></cdg-icon>

<style>
.sample-pharagraph {
  color: var(--danger);
}
</style>
<p class="sample-pharagraph">
    This pharagraph have danger color
    <cdg-icon name="user"></cdg-icon>
</p>
