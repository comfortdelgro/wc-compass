Textarea is used for inputting multiple lines of text.

Default

  <input class="cdg-input" placeholder="Type something..." />

  <input
  class="cdg-input"
  placeholder="Type something..."
  disabled
  value="You cannot input me"
/>

With label and customize width

  <label class="cdg-input-group" style="width: 320px">
  <span class="cdg-input-label">Input me</span>
  <input class="cdg-input" placeholder="Type something..." />
</label>

With helper message

  <label class="cdg-input-group" style="width: 320px">
  <span class="cdg-input-label">Input me</span>
  <input class="cdg-input" placeholder="Type something..." />
  <span class="cdg-helper-message">Ex: It's awesome</span>
</label>

  <label class="cdg-input-group" style="width: 320px">
  <span class="cdg-input-label">Disabled input field</span>
  <input
    class="cdg-input"
    placeholder="Type something..."
    disabled
    value="You cannot input me"
  />
  <span class="cdg-helper-message">Ex: It's awesome</span>
</label>

Error field and match parent width 

  <label class="cdg-input-group error" style="width: 100%">

  <input class="cdg-input" placeholder="Please input me..." />

</label>
