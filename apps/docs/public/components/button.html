Buttons communicate actions that users can take to trigger events.

Usage

  <button
    class="cdg-button {'' | primary | danger | secondary | ghost}"
    size="{'' | large | small}"
    disabled
    onclick="alert('click')">
    // Left icon
    <cdg-icon name="callCenter"></cdg-icon>
    Button text
    // Right icon
    <cdg-icon name="arrowDown"></cdg-icon>
    // Or replace everything by
    <cdg-inline-loading size="small"></cdg-inline-loading>
</button>
