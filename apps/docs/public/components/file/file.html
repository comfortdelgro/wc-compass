Files is used to select files and show feedback upon uploading a file. It can also be used when a file is provided by the system and can be downloaded by the user.

Browse file
 
    <cdg-file>
    <div class="cdg-button">Browse file</div>
    <input type="file" class="cdg-file-input" />
</cdg-file>
Upload files 
    <cdg-file allow-drop id="uploadFileHorizontal">
    <div class="cdg-button upload-button">
        <cdg-icon name="upload" source="host"></cdg-icon>
        Upload files
    </div>
    <input type="file" class="cdg-file-input" multiple />
</cdg-file>
