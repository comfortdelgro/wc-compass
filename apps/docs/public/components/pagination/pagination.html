Pagination should be used:

  • To enable users to focus on a few items at a time.
  • To make it clear thereâ€™s more to explore when ready.
  • For large collections of items.
  <cdg-pagination
  id="simple-pagination"
  current-page="1"
  total="30"
  page-size="10"
></cdg-pagination>
<script>
const simplePaging = document.querySelector('#simple-pagination');

simplePaging.addEventListener('navigate', (data) => {
  simplePaging.currentPage = data.detail;
});
</script>

Why link?
With sever side rendering, we will put the link directly into every single button. So everything works without javascript. Else we will have to handle everything on client using javascript to navigate the page, which is not friendly with sever side rendering architecture.
