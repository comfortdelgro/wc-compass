Props

Name Type Default Description
current number 0
The active index of the slides.
use-arrow boolean false
Using button next/prev as an arrow.
auto-switch boolean false
If true, the slide will automatically switch.
single-center boolean false
If true, the first item will be centered.
scaled boolean false
If true, scale slide item 0.9 times.
onCurrentChange function Emits when the current index is changed.
