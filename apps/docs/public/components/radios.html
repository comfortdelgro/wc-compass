Radios

Radio buttons allow users to exactly select one option from a set.
We reuse the default input radio control to have it own behaviors. Just follow the html structure with cdg-radio class name. Try to navigate focus by keyboard and press arrow keys to change value.
For more implementation documents please refer: mdn web docs

Default

    <label class="cdg-radio">
  <input type="radio" name="sample2" />
  <span>I'm the first one</span>
</label>
<label class="cdg-radio">
  <input type="radio" name="sample2" checked />
  <span>I'm second</span>
</label>
<label class="cdg-radio">
  <input type="radio" name="sample2" />
  <span>I'm the last</span>
</label>

Disabled

  <label class="cdg-radio">
  <input type="radio" name="sample3" checked disabled />
  <span>I'm the first one</span>
</label>
<label class="cdg-radio">
  <input type="radio" name="sample3" disabled />
  <span>I'm two</span>
</label>
<label class="cdg-radio">
  <input type="radio" name="sample3" disabled />
  <span>I'm the last</span>
</label>

Custom content

  <label class="cdg-radio custom-content">
  <input type="radio" name="sample4" checked />
  <div class="radio-content">
    <div class="radio-title">I'm the first one</div>
    <div class="radio-description">
      This is the description to give user more details about this option
    </div>
  </div>
</label>
<label class="cdg-radio custom-content">
  <input type="radio" name="sample4" />
  <div class="radio-content">
    <div class="radio-title">I'm the second</div>
    <div class="radio-description">
      This is the description to give user more details about this option and
      longer than the other radios of this demo
    </div>
  </div>
</label>
<label class="cdg-radio custom-content">
  <input type="radio" name="sample4" />

    <div class="radio-title">I'm the last one</div>
    <div class="radio-description">This is the description</div>
  </div>
</label>
