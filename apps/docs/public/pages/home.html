Usage

Just simply import these files to your html file and copy `images` folder into same level of html file to host our icons. 

    ...
<head>
...
    <link rel="stylesheet" href="./styles/main.bundle.css" />
...
    <!--Add this script before closing body tag-->
    <script defer src="./js/main.bundle.js"></script>
...
</body>
