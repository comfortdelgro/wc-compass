Welcome!

This is guideline page for ComfortDelGro's Workbench Design System library. Which a set of components that created using web component and can be used to implement user experiences consistent with ComfortDelGro's design principles. These components can work without any other library like jQuery, Angular, VueJS, React and can run on most of modern browsers. Refer Workbench Design System

Usage

There are three parts of the package: CSS, Javascript, and Icon or Image resouce. We are fetching icon and images from cdn by default.

To download the package or setting up our project, please refer github repo: comfortdelgro / wc-compass

CDN Links

To let our components and layout work on your project, just use these cdn links: 

    ...
<head>
...
    <link rel="stylesheet" href="https://unpkg.com/@comfortdelgro/wc-compass@latest/dist/main.min.css" />
...
    <!--Add this script before closing body tag-->
    <script src="https://unpkg.com/@comfortdelgro/wc-compass@latest/dist/main.min.js"></script>
...
</body>

Remember that you can change @latest to @{any version}, for example: ...wc-compass@0.0.1/dist...

Host

Our you can also download and host resouces on your sever:

    ...
<head>
...
    <link rel="stylesheet" href="./styles/main.bundle.css" />
...
    <!--Add this script before closing body tag-->
    <script src="./js/main.bundle.js"></script>
...
</body>

Use only css

If you don't want to use our default scripts, or you want it to be lighter and use only our css for any purpose. You can just inject our css into your project and follow our html structure. Style of the design system will also work.
