Wizards

Wizards displays progress through a sequence by breaking it up into multiple steps.

Default

    <cdg-wizards
    steps="General,Additional,Education,Personal,Review"
    current="3"
></cdg-wizards>

<script>
    // Add these if you want to active the step that user click on
    const wizards = document.querySelector('cdg-wizards');
    wizards.addEventListener('switchStep', (event) => {
    console.log(event.detail);
    wizards.current = event.detail.index;
    });
</script>
