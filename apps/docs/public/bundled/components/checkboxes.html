Checkboxes are input controls for selecting more than one option from a list of related items.
We reuse the default input checkbox control to have it own behaviors. Just follow the html structure with cdg-checkbox class name.
For more implementation documents please refer: mdn web docs 

    <div class="cdg-form-row">
  <label is="cdg-checkbox" checked>
    <input type="checkbox" />
  </label>
  <label class="rounded" is="cdg-checkbox" checked>
    <input type="checkbox" />
  </label>
</div>
