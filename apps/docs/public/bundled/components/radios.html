Radio buttons allow users to exactly select one option from a set.
We reuse the default input radio control to have it own behaviors. Just follow the html structure with cdg-radio class name. Try to navigate focus by keyboard and press arrow keys to change value.
For more implementation documents please refer: mdn web docs 

    <div class="cdg-form-row row-wrap">
  <label is="cdg-radio">
    <input type="radio" name="sample2" />
    <span>I'm the first one</span>
  </label>
  <label is="cdg-radio" checked>
    <input type="radio" name="sample2" />
    <span>I'm second</span>
  </label>
  <label is="cdg-radio">
    <input type="radio" name="sample2" />
    <span>I'm the last</span>
  </label>
</div>
