Datepicker

Calendar is used as a date picker to let user select a date or a range of dates.

Default

  
<div>
  <cdg-datepicker start-title="Single Datepicker" />
</div>

Set default date

  
<div>
  <cdg-datepicker start-title="Single Datepicker" date="2023-03-08" />
</div>

Disabled

  
<div style="display: flex; flex-direction: column; gap: 20px">
  <div>
    <cdg-datepicker
      class="diabled-datepicker"
      start-title="Disabled"
      disabled="true"
    />
  </div>
  <div>
    <cdg-datepicker
      class="diabled-datepicker"
      start-title="Start"
      end-title="End"
      disabled="true"
      double
    />
  </div>
</div>

Datepicker with min&max

  
<div>
  <cdg-datepicker
    start-title="min&max"
    min="2023-03-08"
    max="2023-03-15"
  />
</div>

Start & end inputs

  
<div>
  <cdg-datepicker
    double
    start-title="Start"
    end-title="End"
    format="DD-MM-YYYY"
  />
</div>

Start & end with default values

  
<div>
  <cdg-datepicker
    double
    start-title="Start"
    end-title="End"
    start-date="2023-03-08"
    end-date="2023-03-15"
  />
</div>

Format date with DD-MM-YYYY

  
<div>
  <cdg-datepicker start-title="Single Datepicker" format="DD-MM-YYYY" />
</div>
