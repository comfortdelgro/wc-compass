Dropdowns allow users to choose from a list of options in a limited space. The list of options can change based on the context.

Single dropdown

  
<cdg-dropdown placeholder="Select an item" header-title="Filter title">
  <cdg-dropdown-option value="1">
    Dropdown 1
  </cdg-dropdown-option>
  <cdg-dropdown-option value="2" current-color="black">
    Dropdown 2
  </cdg-dropdown-option>
  <cdg-dropdown-option value="3">
    Dropdown 3
  </cdg-dropdown-option>
</cdg-dropdown>

Disabled dropdown

  
<label class="cdg-input-group">
  <span class="cdg-input-label">Disabled dropdown</span>
  <cdg-dropdown
    placeholder="Select an item"
    header-title="Filter title"
    disabled
    id="disabledDropdown"
  >
    <cdg-dropdown-option value="1"> Dropdown 1 </cdg-dropdown-option>
    <cdg-dropdown-option value="2" current-color="black" selected="true">
      Dropdown 2
    </cdg-dropdown-option>
    <cdg-dropdown-option value="3"> Dropdown 3 </cdg-dropdown-option>
  </cdg-dropdown>
</label>
