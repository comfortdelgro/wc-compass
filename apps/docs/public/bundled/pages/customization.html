Style

Our components has built with simply css selector with prefix cdg for you to use with any other libraries without impacts. So to override our style, just make your css selector 1 more level complicated.
Remember to not use !important flag, this will pulls you into other problems.

For example:

    /** Our CSS **/
.cdg-card {
    border-radius: 4px;
    overflow: hidden;
    display: flex;
    flex-direction: column;
    background: var(--background);
    color: var(--grey100);
}

/** To override **/
.cdg-card.customized {
    /** Any override css here **/
    border-radius: 8px;
    color: var(--grey800);
}

Component

We don't expect you to customize our component. But in case you need to quickly override our behaviors, you also can do that. And if your customization will need to reuse every where don't hesitate to contact us to make your customization as common behaviors or layouts
To customize our components, you can import and extend our component to build a new customized component.
